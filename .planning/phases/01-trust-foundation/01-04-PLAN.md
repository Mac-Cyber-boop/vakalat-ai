    Create src/verification/citation_gate.py implementing citation blocking:

    1. Import dependencies:
       - re (for citation extraction)

       - pydantic (BaseModel, Field)
       - src.verification.citation_verifier (CitationVerifier)
       - src.verification.section_validator (SectionValidator)
       - src.verification.audit (get_audit_logger, AuditEvent)
       - datetime

    2. Create FilteredCitations model:
       - verified: List[str] = [] (citations that passed verification)
       - blocked: List[str] = [] (citations that failed verification)
       - blocked_reasons: dict[str, str] = {} (citation -> reason mapping)

    3. Create CitationGate class:

       __init__(self, citation_verifier: CitationVerifier, section_validator: Optional[SectionValidator] = None):
       - Store verifier and validator references
       - Initialize audit logger

       _extract_case_citations(self, text: str) -> List[str]:
       - Use regex to extract case citations from text
       - Pattern: Match formats like "Arnesh Kumar vs State of Bihar", "(2014) 8 SCC 273"
       - Common patterns:
         - r'([A-Z][a-z]+ (?:vs?\.?|versus) [A-Z][^,\n]+)'  # "X vs Y" or "X v. Y"
         - r'\(\d{4}\)\s*\d+\s*SCC\s*\d+'  # "(2014) 8 SCC 273"
         - r'AIR\s*\d{4}\s*SC\s*\d+'  # "AIR 2014 SC 123"
       - Return list of extracted citations

       _extract_statute_citations(self, text: str) -> List[Tuple[str, str]]:
       - Use regex to extract statute citations
       - Pattern: "Section 302, IPC" or "Section 103 of BNS" or "IPC Section 420"
       - Return list of (act_name, section) tuples

       _parse_case_citation(self, citation: str) -> Tuple[str, int, Optional[str]]:
       - Parse citation into (case_name, year, formal_citation)
       - Extract year from citation text
       - Return tuple for verification

       filter_citations(self, text: str) -> FilteredCitations:
       - Extract all case citations from text
       - For each citation:
         - Parse into components
         - Call citation_verifier.verify_case_citation()
         - If VERIFIED: add to verified list
         - If UNVERIFIED: add to blocked list with reason
         - Log each verification attempt
       - Return FilteredCitations result

       filter_statute_citations(self, text: str) -> FilteredCitations:
       - Extract all statute citations from text
       - For each (act_name, section):
         - Call section_validator.validate_section() if validator provided
         - If VERIFIED or OUTDATED: add to verified (outdated is still valid citation)
         - If UNVERIFIED: add to blocked
       - Return FilteredCitations result

       sanitize_output(self, text: str, filtered: FilteredCitations) -> str:
       - Remove blocked citations from text
       - Per CONTEXT.md: "omit and continue" - don't fail, just remove
       - Return sanitized text with blocked citations removed
       - Optionally add footnote: "Note: Some citations could not be verified against our database."

    4. Handle edge cases:
       - Empty text returns empty FilteredCitations
       - Malformed citations logged but not crashed
       - Multiple citations on same line handled correctly

          ```python
          @app.post("/draft")
          async def draft(req: DraftRequest):
              if not vector_db: raise HTTPException(500, "Database Connection Failed")

              ctx = "\n".join([d.page_content for d in vector_db.similarity_search(
                  f"limitation mandatory requirements {req.doc_type}", k=8)])

              res = await (DRAFT_PROMPT | llm | StrOutputParser()).ainvoke({
                  "doc_type": req.doc_type,
                  "details": req.details,
                  "context": ctx,
                  "current_date": str(date.today())
              })

              draft_html = res.replace("```html", "").replace("```", "").strip()

              # Citation verification
              verification_applied = False
              blocked_count = 0

              if citation_gate:
                  filtered = citation_gate.filter_citations(draft_html)
                  if filtered.blocked:
                      blocked_count = len(filtered.blocked)
                      draft_html = citation_gate.sanitize_output(draft_html, filtered)
                      verification_applied = True

              return {
                  "draft": draft_html,
                  "citations_verified": verification_applied,
                  "citations_blocked": blocked_count
              }
          ```
