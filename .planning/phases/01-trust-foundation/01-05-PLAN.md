    3. Create DetectionResult model:
       - outdated_references: List[OutdatedReference]
       - has_outdated: bool (True if any outdated references found)
       - summary: str (e.g., "Found 3 references to old legal codes")

    4. Create OutdatedCodeDetector class:

       OLD_CODE_PATTERNS = [
           r'(?:Section|S\.?|Sec\.?)\s*(\d+[A-Za-z]?)\s*(?:,|\s+of\s+|\s+)?\s*(IPC|Indian Penal Code)',
           r'(?:Section|S\.?|Sec\.?)\s*(\d+[A-Za-z]?)\s*(?:,|\s+of\s+|\s+)?\s*(CrPC|Cr\.P\.C\.?|Code of Criminal Procedure)',
           r'(?:Section|S\.?|Sec\.?)\s*(\d+[A-Za-z]?)\s*(?:,|\s+of\s+|\s+)?\s*(IEA|Indian Evidence Act|Evidence Act)',
           r'(IPC|CrPC|IEA)\s+(?:Section|S\.?|Sec\.?)\s*(\d+[A-Za-z]?)',  # Reversed: "IPC Section 302"
       ]

       __init__(self, code_mapper: Optional[LegalCodeMapper] = None):
       - Initialize code_mapper (create if not provided)

       detect_outdated(self, text: str) -> DetectionResult:
       - Apply all patterns to find old code references
       - For each match:
         - Extract old_code and old_section
         - Call code_mapper.map_section() to get new equivalent
         - Create OutdatedReference with suggestion
       - Return DetectionResult with all findings

       _format_suggestion(self, old_code: str, old_section: str, new_code: Optional[str], new_section: Optional[str]) -> str:
       - Format user-friendly suggestion message
       - Example: "Section 302, IPC corresponds to Section 103, BNS (effective July 1, 2024)"
       - If no mapping: "Section {old_section}, {old_code} - please verify current equivalent"

       annotate_text(self, text: str, result: DetectionResult) -> str:
       - Insert inline annotations for outdated references
       - Example: "Section 302, IPC [Note: Now Section 103, BNS]"
       - Preserve original text, add annotations

       get_suggestions_html(self, result: DetectionResult) -> str:
       - Generate HTML snippet with all suggestions
       - For use in draft endpoint response
       - Format as an info box with list of updates

       e. Add new /verify-citation endpoint for explicit verification:
          ```python
          class VerifyCitationRequest(BaseModel):
              citation_type: str = Field(description="'case' or 'statute'")
              case_name: Optional[str] = None
              year: Optional[int] = None
              act_name: Optional[str] = None
              section: Optional[str] = None

          @app.post("/verify-citation")
          async def verify_citation(req: VerifyCitationRequest):
              if req.citation_type == "case":
                  if not citation_verifier:
                      raise HTTPException(500, "Citation verifier not available")
                  result = citation_verifier.verify_case_citation(req.case_name, req.year)
                  return result.model_dump()
              elif req.citation_type == "statute":
                  if not section_validator:
                      raise HTTPException(500, "Section validator not available")
                  result = section_validator.validate_section(req.act_name, req.section)
                  return result.model_dump()
              else:
                  raise HTTPException(400, "citation_type must be 'case' or 'statute'")
          ```
