       - src.verification.audit (get_audit_logger, AuditEvent, log_verification_attempt)

    2. Create CitationVerifier class:

       __init__(self, vector_store: PineconeVectorStore):
       - Store vector_store reference
       - Initialize audit logger: self.logger = get_audit_logger("citation_verifier")
       - Initialize cache for verified citations (use @lru_cache on method or manual dict)

       verify_case_citation(self, case_name: str, year: int, citation: Optional[str] = None) -> VerificationResult:
       - Start timer: start = time.perf_counter()
       - Normalize case_name to lowercase for matching
       - Build Pinecone filter for exact metadata match:
         ```python
         filter_dict = {
             "$and": [
                 {"source_type": {"$eq": "case_law"}},
                 # Try case_name or title field based on existing metadata
             ]
         }
         ```
       - Query vector_store.similarity_search(query=case_name, k=3, filter=filter_dict)
       - Calculate duration_ms = int((time.perf_counter() - start) * 1000)
       - If results found:
         - Return VerificationResult(status=VERIFIED, source_id=results[0].metadata.get("source_id"), confidence=1.0, verification_time_ms=duration_ms)
       - If not found:
         - Return VerificationResult(status=UNVERIFIED, reason="Case not found in legal database", verification_time_ms=duration_ms)
       - Log attempt using audit logger before returning

       verify_from_input(self, input: CaseCitationInput) -> VerificationResult:
       - Convenience wrapper: return self.verify_case_citation(input.case_name, input.year, input.citation)

    3. Handle the metadata field variation:
       - Existing ingestion uses "title" for judgments (from ingest_master.py)
       - Try both "case_name_normalized" and "title" fields
       - If neither works with filter, fall back to semantic search with post-filtering

    4. Update src/verification/__init__.py:
       - Add import: from src.verification.citation_verifier import CitationVerifier
       - Add to __all__: "CitationVerifier"

    IMPORTANT: Per CONTEXT.md, exact match is required. Do NOT use fuzzy matching or semantic similarity scoring to "approximate" matches. If the case is not found with exact metadata filter, return UNVERIFIED.
  </action>
