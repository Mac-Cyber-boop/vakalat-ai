    2. Create src/templates/schemas.py with Pydantic models following 02-RESEARCH.md patterns:

       a. DocumentType enum (str, Enum):
          - BAIL_APPLICATION = "bail_application"
          - LEGAL_NOTICE = "legal_notice"
          - AFFIDAVIT = "affidavit"
          - PETITION = "petition"

       b. CourtLevel enum (str, Enum):
          - SUPREME_COURT = "supreme_court"
          - HIGH_COURT = "high_court"
          - DISTRICT_COURT = "district_court"

       c. TemplateMetadata model:
          - doc_type: DocumentType
          - court_level: CourtLevel
          - version: str = Field(description="Semantic version e.g. 1.0.0")
          - name: str
          - description: str

       d. FormattingRequirements model (Supreme Court Rules 2013 defaults):
          - paper_size: str = Field(default="A4", description="29.7cm X 21cm")
          - font: str = Field(default="Times New Roman")
          - font_size: int = Field(default=14, ge=12, le=16)
          - line_spacing: float = Field(default=1.5)
          - margin_left_right: str = Field(default="4cm")
          - margin_top_bottom: str = Field(default="2cm")
          - paper_quality: str = Field(default="75 GSM")
          - double_sided: bool = Field(default=True)

       e. TemplateField model:
          - field_name: str = Field(min_length=1, max_length=100)
          - field_type: str = Field(description="text|date|party|case_number|relief|court")
          - required: bool
          - label: str
          - description: str
          - example: Optional[str] = None
          - validation_regex: Optional[str] = None

          Add @field_validator for field_type to ensure it's one of the allowed types.
          Add @field_validator for field_name to ensure snake_case.

       f. LegalTemplate model:
          - metadata: TemplateMetadata
          - formatting: FormattingRequirements
          - required_fields: List[TemplateField]
          - optional_fields: List[TemplateField] = []
          - template_content: str = Field(default="", description="Template structure for future Jinja2 use")
          - created_at: str = Field(default_factory=lambda: datetime.now().isoformat())
          - updated_at: str = Field(default_factory=lambda: datetime.now().isoformat())

    3. Update src/templates/__init__.py to export all models:
       - DocumentType, CourtLevel, FormattingRequirements, TemplateField, LegalTemplate, TemplateMetadata

    Follow existing api.py Pydantic conventions: use Field with descriptions, follow naming patterns.

  <action>
    Create src/templates/storage.py implementing the TemplateRepository class:

    1. TemplateRepository class:
       - __init__(self, data_dir: Path = None):
         Default to src/templates/data/ relative to this file.
         Store data_dir as instance variable.
         Create data_dir if it doesn't exist.

       - _get_filename(self, doc_type: DocumentType, court_level: CourtLevel) -> str:
         Return f"{doc_type.value}_{court_level.value}.json"

       - get_template(self, doc_type: DocumentType, court_level: CourtLevel) -> Optional[LegalTemplate]:
         Build filepath from data_dir + filename.
         If file doesn't exist, return None.
         Load JSON, validate with LegalTemplate(**data), return template.
         Handle JSONDecodeError and ValidationError gracefully (log warning, return None).

       - list_templates(self, doc_type: Optional[DocumentType] = None) -> List[LegalTemplate]:
         Glob *.json in data_dir.
         For each file, load and validate as LegalTemplate.
         If doc_type filter provided, only include matching templates.
         Sort by (doc_type, court_level) for consistent ordering.
         Skip invalid files with warning log.

       - save_template(self, template: LegalTemplate) -> Path:
         Generate filename from template.metadata.doc_type and court_level.
         Write template.model_dump() as JSON with indent=2, ensure_ascii=False.
         Return the filepath written.

       - template_exists(self, doc_type: DocumentType, court_level: CourtLevel) -> bool:
         Check if file exists without loading.

    2. Add logging using Python's logging module (consistent with codebase).

    3. Update src/templates/__init__.py to export TemplateRepository.

    Note: The data directory will be empty until Plan 02 creates the template JSON files.
  </action>
