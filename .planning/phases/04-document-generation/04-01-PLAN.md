<task type="auto">
  <name>Task 1: Create fact collection models</name>
  <files>src/generation/models.py</files>
  <action>
Create Pydantic models for structured fact collection. Each model mirrors the required_fields from corresponding templates.

**CRITICAL: Before implementing, verify field alignment with templates:**
1. Read each template JSON file (bail_application_supreme_court.json, legal_notice_supreme_court.json, affidavit_supreme_court.json, petition_supreme_court.json)
2. Extract the `field_name` values from each template's `required_fields` array
3. Ensure each model below has fields matching those template field_names exactly

**BailApplicationFacts** (must match bail_application_*.json required_fields):
- `applicant_name: str` - Field(min_length=3, max_length=200)
- `applicant_address: str` - Field(min_length=10)
- `fir_number: str` - Field(pattern=r"FIR No\. \d+/\d{4}")
- `police_station: str`
- `sections_charged: str`
- `date_of_arrest: date`
- `grounds_for_bail: str` - Field(min_length=50, description="Plain language - LLM converts to formal") [NOTE: template uses grounds_for_bail, not grounds_summary]
- `relief_sought: str`

**LegalNoticeFacts** (must match legal_notice_*.json required_fields):
- `sender_name: str`
- `sender_address: str`
- `recipient_name: str`
- `recipient_address: str`
- `subject_matter: str`
- `facts_summary: str` - Field(min_length=50)
- `demand_or_relief: str`

**AffidavitFacts** (must match affidavit_*.json required_fields):
- `deponent_name: str`
- `deponent_address: str`
- `deponent_occupation: str`
- `statements: list[str]` - Field(min_length=1, description="Numbered statements of fact")
- `purpose: str`

**PetitionFacts** (must match petition_*.json required_fields):
- `petitioner_name: str`
- `petitioner_address: str`
- `respondent_name: str`
- `respondent_address: str`
- `facts_summary: str` - Field(min_length=50)
- `grounds_summary: str` - Field(min_length=50)
- `relief_sought: str`

**Validators to include:**
- `date_of_arrest` cannot be in the future (for BailApplicationFacts)
- `statements` list must have at least one item (for AffidavitFacts)

Use Pydantic v2 syntax with `field_validator` decorator. Follow patterns from `src/templates/schemas.py`.

Include docstrings explaining each model's purpose and linking to corresponding template.
  </action>
