**CRITICAL: Citation integration wiring in _generate_field_content:**
The `_generate_field_content` method must integrate citations as follows:
1. Extract citation strings from `citations` parameter: `citation_strings = [c.formatted_citation for c in citations]`
2. Call `get_generation_prompt(court_level, field_name, citations=citation_strings)` to build system prompt that includes citation integration instructions
3. The `get_generation_prompt` function (from prompts.py) appends citation guidance to the system prompt when citations are provided
4. LLM response will include relevant citations from the list in its generated content

Example implementation:
```python
def _generate_field_content(
    self,
    field_name: str,
    user_input: str,
    court_level: CourtLevel,
    citations: list[CitationRecommendation]
) -> str:
    # Extract citation strings for prompt builder
    citation_strings = [c.formatted_citation for c in citations] if citations else None

    # Build system prompt with citations included
    system_prompt = get_generation_prompt(
        court_level=court_level,
        field_name=field_name,
        citations=citation_strings  # Pass citations to prompt builder
    )

    # Call LLM with system prompt containing citation guidance
    response = self.llm_client.chat.completions.create(
        model="gpt-4o-2024-08-06",
        messages=[
            {"role": "system", "content": system_prompt},
            {"role": "user", "content": user_input}
        ]
    )
    return response.choices[0].message.content
```

**Field type handling:**
- `text`, `relief` fields: Call LLM to expand user input into formal legal language
- `party`, `date`, `case_number`, `court` fields: Use user input directly (no LLM)
