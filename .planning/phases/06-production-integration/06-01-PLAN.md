2. Create `export_to_pdf(content: str, formatting: FormattingRequirements, title: str = "") -> BytesIO` function that:
   - Parses margin strings ("4cm" -> 40mm) from FormattingRequirements
   - Sets margins using set_margins() and set_auto_page_break()

   - Returns BytesIO with seek(0) called (critical: prevents empty file)

- margin_left_right "4cm" -> 40 mm (multiply by 10)
- font "Times New Roman" -> "times" (fpdf2 naming)
- line_spacing 1.5 with font_size 14 -> line_height = 14 * 1.5 * 0.353 = 7.4mm
