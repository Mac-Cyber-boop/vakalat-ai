  <name>Task 1: Create citation data models</name>
  <files>src/citations/models.py</files>
  <action>
Create Pydantic models for citation handling:

1. **CitationBase** - Base model with:
   - `raw_text: str` - Original citation text
   - `citation_type: Literal["case", "statute"]`

2. **CaseCitation(CitationBase)** - For case law:
   - `party_a: str` - First party name
   - `party_b: str` - Second party name (can be "State of X" or "Union of India")
   - `year: int` - Year of judgment
   - `reporter: Optional[Literal["SCC", "AIR", "OTHER"]]` - Reporter name
   - `volume: Optional[int]` - For SCC format
   - `page: Optional[int]` - Page number
   - `court: Optional[str]` - Court code (SC, Del, Bom, etc.)

3. **StatuteCitation(CitationBase)** - For statutes:
   - `section: str` - Section number (may include letters like "302A")
   - `act_name: str` - Full act name
   - `year: Optional[int]` - Year of enactment (1973 for CrPC, etc.)

4. **VerificationBadge** - For UI display:
   - `status: Literal["verified", "unverified", "outdated"]`
   - `icon: str` - Unicode or HTML entity (checkmark, warning, refresh)
   - `tooltip: str` - Hover text explaining status
   - `css_class: str` - CSS class for styling

5. **FormattedCitation** - Complete formatted output:
   - `formatted_text: str` - Properly formatted citation string
   - `badge: VerificationBadge` - Verification status
   - `source_id: Optional[str]` - Database ID if verified
   - `html_output: str` - Property that combines formatted_text + badge HTML

Follow src/verification/models.py patterns:

<task type="auto">
  <name>Task 2: Create citation formatter</name>
  <files>src/citations/formatter.py</files>
  <action>
Create CitationFormatter class with static methods for each citation format:

1. **format_scc(party_a, party_b, year, volume, page) -> str**
   - Output: "Arnesh Kumar vs State of Bihar (2014) 8 SCC 273"
   - Uses "vs" (not "v." or "versus") per Indian convention

2. **format_air(party_a, party_b, year, court, page) -> str**
   - Output: "K.S. Puttaswamy vs Union of India, AIR 2017 SC 4161"
   - court param uses abbreviation (SC, Del, Bom, Cal, Mad, Kar, All)

3. **format_statute(section, act_name, year=None) -> str**
   - Output: "Section 438, Code of Criminal Procedure, 1973"
   - Year is optional (some acts don't include year in citation)

4. **HIGH_COURT_ABBREVIATIONS** - Class constant dict:
   ```python
   {
       "supreme_court": "SC",
       "delhi": "Del", "delhi_high_court": "Del",
       "bombay": "Bom", "bombay_high_court": "Bom",
       "calcutta": "Cal", "calcutta_high_court": "Cal",
       "madras": "Mad", "madras_high_court": "Mad",
       "karnataka": "Kar", "karnataka_high_court": "Kar",
       "allahabad": "All", "allahabad_high_court": "All",
   }
   ```

5. **format_with_badge(citation: CaseCitation|StatuteCitation, badge: VerificationBadge) -> FormattedCitation**
   - Formats citation using appropriate method
   - Attaches verification badge
   - Returns complete FormattedCitation model

6. **get_badge_html(badge: VerificationBadge) -> str**
   - Returns: `<span class="citation-badge {css_class}" title="{tooltip}">{icon}</span>`

Badge icons (use Unicode for portability):
- verified: checkmark U+2713
- unverified: warning U+26A0
- outdated: refresh U+21BB
