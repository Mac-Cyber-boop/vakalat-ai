<task type="auto">
  <name>Task 2: Create CitationRecommender class</name>
  <files>src/citations/recommender.py</files>
  <action>
Create the main recommender class:

1. **Constructor**:
   ```python
   def __init__(
       self,
       vector_store: PineconeVectorStore,
       citation_verifier: Optional[CitationVerifier] = None
   ):
       self.retriever = PrecedentRetriever(vector_store)
       self.citation_verifier = citation_verifier
       self.formatter = CitationFormatter
   ```

2. **recommend_precedents(legal_issue: str, filing_court: str = "supreme_court", top_k: int = 5) -> list[CitationRecommendation]**:
   - Step 1: Call retriever.retrieve_precedents(legal_issue, filing_court, top_k)
   - Step 2: For each retrieved precedent:
     a. Verify with citation_verifier if available
     b. Format citation using CitationFormatter (prefer SCC format)
     c. Create CitationRecommendation with badge
   - Step 3: Return list sorted by relevance_score

3. **_verify_precedent(precedent: RetrievedPrecedent) -> tuple[str, Optional[str]]**:
   - Returns (status, source_id)
   - If citation_verifier is None, return ("unverified", None)
   - Otherwise call citation_verifier.verify_case_citation(case_name, year)
   - Return ("verified", source_id) or ("unverified", None)

4. **_format_precedent(precedent: RetrievedPrecedent) -> str**:
   - Extract party names from case_name (split on "vs", "v.", "versus")
   - If SCC metadata available (volume, page), use format_scc
   - If AIR metadata available (court, page), use format_air
   - Otherwise, return raw case_name with year: "Case Name (Year)"

5. **_parse_case_parties(case_name: str) -> tuple[str, str]**:
   - Split on "vs", "v.", "versus" (case-insensitive)
   - Return (party_a.strip(), party_b.strip())
   - Handle edge cases like "State of X & Anr"
